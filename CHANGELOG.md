## [2.0.0] - 2026-04-22

3-tab 확장 릴리스. 보험금융손익 · 예실차 탭을 추가하고, 마감년월 선택 기준 전 탭 동기화 및 챗봇 다중 데이터셋 지원을 도입.

### Added
- **보험금융손익 탭 (financial.js)**
  - 섹션 1/2: NP·IDP 이자부리 보험금융비용 (상품유형 × 월)
  - 섹션 3: VFA 이자부리/위험경감 (상품 × 구분 × 월)
  - 섹션 4: 부담이자율 — 상품별 `rawData2_2` + 합계/총합 `rawData2_1`
  - 섹션 5: 공시이율 예실차 (OCI)
- **예실차 탭 (variance.js)**
  - 섹션 1: 지급보험금 예실차(당월) — 코호트 매핑(999991→~2018 … 202501→2025) + Total(Expected/Actual/Variance)
  - 섹션 1-1: 상품군 × (예/실/차) × 최근 12개월 + 합계
  - 섹션 2: 사업비 예실차(당월) — 유지비/신계약비(PL)/신계약비(CSM) + 합계
  - 섹션 2-1: 구분 × (예/실/차) × 최근 12개월 + 합계
  - 섹션 3: 기타 현금흐름 예실차(당월) — 수입보험료(보험료)/투자요소 보험금(보험금(CSM))/약관대출/기타지급금(계약자배당) + Total + 합계
  - 섹션 3-1: 구분 × (예/실/차) × 최근 12개월 + 합계
  - 예실차 전용 상품유형 정렬(`VAR_PRODUCT_ORDER`)
- **헤더 마감년월 콤보박스**
  - 데이터의 모든 마감년월을 내림차순 옵션으로 제공
  - 선택 변경 시 보험손익 · 보험금융손익 · 예실차 탭이 모두 재렌더
  - 보험손익 1.당월=선택월 / 2.당해누적=연도 시작~선택월 / 최근 12개월=선택월 종료 12개월
  - 예실차 당월 표 3개에 선택월 태그(`YYYY.MM`) 표시
- **챗봇 다중 데이터셋 지원**
  - 3개 탭 스키마 설명 + 월별 요약 CSV 자동 포함 (보험손익, 보험금융비용/부담이자율, 예실차)
  - 선택월 × 상품유형 × 구분 × 예실구분 상세 행 추가
  - 활성 탭 자동 감지 + 현재 화면 표 스냅샷을 프롬프트에 포함
  - 답변을 간결 모드(2~4문장)로 지침 변경, `result` 이벤트 중복 제거
- **챗봇 창 리사이즈** — 우측 하단 핸들로 자유 크기 조절 (320×320 ~ 95vw×90vh)

### Changed
- 모든 표에서 `그외` 행 제외
- 음수만 빨간색, 양수·0은 기본색으로 통일
- 상품유형 기본 정렬 순서에서 `(무)` 그룹을 `(유)` 그룹 앞으로 이동
- `.variance-grid` 2열 폭을 `minmax(0, 1fr)`로 고정해 좌/우 표 폭 정렬
- 예실차 탭 Total 그룹 헤더 가운데 정렬
- 보험손익 탭 행 간격 축소 (패딩 · line-height)
- 보험금융손익 탭 제목 `(단월)` → `(당월)` 통일
- 모든 탭의 표 세로 스크롤 제거 (전체 행 펼쳐 보기)
- 테이블 wrapper `max-height: 520px` 제거

### Fixed
- 1.당월 표의 period-tag 복구 (선택월 반영)
- 챗봇 스트리밍 응답 중복 출력 문제 (assistant 이벤트만 사용, `result` 이벤트 무시)

### Assets
- `dashbord_sample/샘플대시보드3_1.jpg` — 예실차 섹션 3/3-1 참조 이미지
- `dashbord_sample/예실차 상품유형 순서.PNG` — 예실차 탭 상품 정렬 참조

## [1.0.0] - 2026-04-20

최초 릴리스 버전. 샘플 JPG 레이아웃과 미래에셋 공식 디지털 가이드라인을 적용한 베이스라인.

### Added
- **대시보드 레이아웃 (3 섹션)**
  - `1.당월` P&L 테이블 — 회계모형(NP/IDP/VFA) × 구분/구분2 집계
  - `2.당해 누적` P&L 테이블 — 연간 누적 집계
  - `3.최근 12개월 월별 추이` — 라인 차트 + 월별 데이터 테이블
- **P&L 계산 로직**
  - 보험수익 → 보험서비스비용 → 보험손익(차감전) → 간접사업비 → 보험손익(차감후)
- **데이터 필터** — 회계모형, 구분 드롭다운
- **챗봇** — 플로팅 버튼형 Q&A 인터페이스

### Design
- **미래에셋 공식 디지털 가이드라인(2020.04) 적용**
  - 주색상: Orange `#F58220`, Blue `#043B72`
  - 보조: Orange Light `#FAAF72`, Footer BG `#ECEFF4`
  - 서체: Spoqa Han Sans Neo (CDN)
  - 헤더: Type A (오렌지 배경 + 화이트 로고)
  - 푸터: Blue 배경 + 화이트 로고 + 고객센터 정보
  - 드래그 선택: 텍스트 `#FFFFFF` / 배경 `#F58220`
- **로고 자산** — `assets/logo/`에 미래에셋생명 공식 로고 포함

### Tech Stack
- HTML / CSS / Vanilla JavaScript
- Chart.js 4.4.7 (CDN)
- Spoqa Han Sans Neo (CDN)
